# wallpaper_change
1.首先爬取了https://iw233.cn/ 的图库具体见mirlkoi_spider.py<br>
2.为了适应屏幕每次从wallpaper中选取8张图片按照1024*1556的比例拼接在一起<br>
3.运行change.py，每30min更换一次壁纸<br>
